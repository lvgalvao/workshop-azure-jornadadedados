Workshop de Azure para Engenharia de Dados

Descubra o poder do Microsoft Azure neste workshop imersivo! Você aprenderá desde conceitos básicos de cloud computing até a implementação prática de dois projetos completos de engenharia de dados. Vamos explorar máquinas virtuais, arquiteturas serverless, ingestão de APIs, armazenamento em Blob Storage, bancos de dados gerenciados e processamento de dados. Tudo isso em um único dia de aprendizado intensivo e dinâmico!

Primeiro Projeto do Workshop

Este é o nosso primeiro projeto dentro do Workshop de Azure para Engenharia de Dados. Neste projeto inicial, você terá a oportunidade de aplicar os conceitos aprendidos de forma prática, desenvolvendo soluções que utilizam diversos serviços da Azure. O foco será na criação de uma infraestrutura robusta e escalável para suportar aplicações de engenharia de dados, garantindo eficiência e segurança.

Durante este projeto, abordaremos tópicos como:

  • Configuração de Máquinas Virtuais no Azure
  • Implementação de Arquiteturas Serverless
  • Ingestão e Processamento de APIs
  • Gerenciamento de Armazenamento com Blob Storage
  • Uso de Bancos de Dados Gerenciados

Este projeto servirá como a base para os demais que serão desenvolvidos ao longo do workshop, proporcionando uma compreensão sólida das ferramentas e práticas essenciais em engenharia de dados na nuvem.

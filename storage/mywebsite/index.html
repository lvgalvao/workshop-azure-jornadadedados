Workshop de Azure para Engenharia de Dados

Chegou a hora de turbinar o que construímos na terça-feira! Nesse workshop de Azure, vamos pegar o que criamos e migrar tudo para as tecnologias do Microsoft Azure. Com isso, você vai entender como usar a nuvem para escalar e otimizar soluções de engenharia de dados de forma prática e eficiente.

O que vamos fazer no Workshop

Vamos trabalhar juntos na migração e implementação prática de dois projetos usando os serviços mais poderosos do Azure. O foco será em entender como essas ferramentas podem transformar seu dia a dia na área de dados.

No primeiro projeto, vamos montar toda a infraestrutura, incluindo:

  • Configuração de Máquinas Virtuais no Azure
  • Implementação de Arquiteturas Serverless
  • Ingestão e Processamento de APIs
  • Gerenciamento de Armazenamento com Blob Storage
  • Uso de Bancos de Dados Gerenciados

E para fechar com chave de ouro, você vai ver como escalar e gerenciar os recursos na nuvem para entregar soluções rápidas e seguras.
